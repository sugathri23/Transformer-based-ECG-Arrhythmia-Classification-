# Transformer-based-ECG-Arrhythmia-Classification-
This project presents a deep learning-based approach for automatic classification of cardiac arrhythmias using ECG (Electrocardiogram) signals. The model is designed to assist in early detection of heart abnormalities by analysing ECG waveforms and classifying them into clinically relevant categories.

Dataset:
The model is trained and evaluated on the MIT-BIH Arrhythmia Dataset:
https://www.kaggle.com/datasets/protobioengineering/mit-bih-arrhythmia-database-modern-2023
